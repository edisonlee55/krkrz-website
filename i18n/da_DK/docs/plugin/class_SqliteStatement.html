SqliteStatement

Sqliteのステートメントを扱うクラス

メンバ

メソッド
bind (パラメータのバインド @parma params バインドするパラメータ(配列または辞書) )
bindAt (パラメータのバインド @parma value 値 )
close (ステートを閉じる )
exec (ステートメント実行 )
get (結果の取得。 )
getName (カラム名の取得 )
getType (カラムの型の取得 )
isNull (カラムのNULL判定 )
open (新規ステートを開く )
reset (バインド状態のリセット )
step (ステートメントステップ実行 )
プロパティ
columnCount (カラム数(読み込み専用) )
count (データ数(読み込み専用) step の後設定されます )
sql (元SQL(読み込み専用) )
定数
SQLITE_INTEGER (カラムの型識別用 )
