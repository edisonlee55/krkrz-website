SqliteThread

sqliteのスレッド実行サポート

メンバ

コンストラクタ
SqliteThread
メソッド
abort (実行処理を中止させます。 )
onProgress (更新処理経過 )
onStateChange (状態変更 )
select (検索処理をバックグランドで実行する。 )
update (更新処理をバックグランドでトランザクション実行する。 )
プロパティ
errorCode (エラーコード )
progressUpdateCount (onProgress で呼び出す頻度指定 )
selectResult (select取得データ(配列) )
state (実行ステート )
定数
DONE (処理完了 )
INIT (初期状態 )
WORKING (処理中 )
