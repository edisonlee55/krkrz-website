コンストラクタ

メソッド
drawContent (ラベルの内容を描画。サイズを再設定 )

onFontFaceModified (フォントフェイスが変更されたら再描画 )
onFontHeightModified (フォントヘイトが変更されたら再描画 )

maxLabelWidth (最大ラベル幅が変更されたら再描画 )
