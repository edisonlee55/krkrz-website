プロパティ
asText (色を16進数のテキストとして表現 )
hasAlpha (αの有無 色の表現にα値を含むかどうかを指定します。 α値を含む場合は0xAARRGGBB形式、 含まない場合は0xRRGGBB形式の整数で色を指定します。 デフォルト設定はfalse(αを含まない)です。 )
pickerDict (カラーピッカーの辞書 )
pickerName (カラーピッカーの名前 )
value (カラーの値 hasAlpha = trueの時は 0xAARRGGBB、 hasAlpha = falseの時は0xRRGGBB形式の整数で色を指定します。 )
