KWidget.onFileDrop

機能/意味
外部からファイルがドロップされた デフォルト動作では、親のonFileDropを呼ぶ。 順ぐりに、子から親へメッセージがチェーンされ、 どこかのウィジェットが関数をオーバーライドして 処理を行うまでチェーンが続く
タイプ
KWidgetクラスのメソッド
構文
onFileDrop(x, y, files, shift)
引数
x ファイルがドロップされた時のマウスのX座標
y ファイルがドロップされた時のマウスのY座標
files ドロップされたファイル名が格納された配列(Array)オブジェクトです。
shift キーが押された時に同時に押されていたシフト系のキーの状態です。
戻り値
なし (void)
説明
