KVirtualWidget.operateStretch

機能/意味
拡大縮小演算合成
タイプ
KVirtualWidgetクラスのメソッド
構文
operateStretch(left, top, width, height, src, sleft, stop, swidth, sheight, mode, opa, type)
引数
left 　重ね合わせ先の矩形の左端位置を ( 重ね合わせ先レイヤの画像位置における ) ピクセル単位で、仮想座標系において指定します。
top 　重ね合わせ先の矩形の上端位置を ( 重ね合わせ先レイヤの画像位置における ) ピクセル単位で、仮想座標系において指定します。
width 　重ね合わせ先の矩形の横幅を ( 重ね合わせ先レイヤの画像位置における ) ピクセル単位で指定します。
height 　重ね合わせ先の矩形の縦幅を ( 重ね合わせ先レイヤの画像位置における ) ピクセル単位で指定します。
src 　重ね合わせ元のレイヤオブジェクトを指定します。
sleft 　重ね合わせる矩形の左端位置を ( 重ね合わせ元レイヤの画像位置における ) ピクセル単位で指定します。
stop 　重ね合わせる矩形の上端位置を ( 重ね合わせ元レイヤの画像位置における ) ピクセル単位で指定します。
swidth 　重ね合わせる矩形の横幅を ( 重ね合わせ元レイヤの画像位置における ) ピクセル単位で指定します。
sheight 　重ね合わせる矩形の縦幅を ( 重ね合わせ元レイヤの画像位置における ) ピクセル単位で指定します。
mode 　演算のモードを指定します。
opa 　演算の強度 ( 0 〜 255 ) を指定します。
type 　拡大縮小のタイプを指定します。
戻り値
なし (void)
説明
