ActiveX

ActiveX を操作できるオブジェクトを生成します。
ActiveX が IDispatch で持つメソッドを直接呼び出すことができます。

メンバ

コンストラクタ
ActiveX
メソッド
onCreate (ウインドウ生成完了後に呼び出されるイベント )
onDestroy (ウインドウ破棄前に呼び出されるイベント )
setExternalUI (拡張UI指定 )
setPos (座標指定 )
setSize (サイズ指定 )
プロパティ
height (縦幅 )
isValidWindow (ウインドウが有効かどうか(読み込み専用) )
left (左端座標 )
top (上端座標 )
visible (表示状態 )
width (横幅 )
