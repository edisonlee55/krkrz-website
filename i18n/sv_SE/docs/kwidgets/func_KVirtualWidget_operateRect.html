KVirtualWidget.operateRect

機能/意味
矩形演算合成
タイプ
KVirtualWidgetクラスのメソッド
構文
operateRect(left, top, src, sleft, stop, swidth, sheight, mode, opa)
引数
vleft 　left 　演算先の矩形の左端位置を ( 演算先レイヤの画像位置における ) ピクセル単位で、仮想座標系において指定します。
vleft 　top 　演算先の矩形の上端位置を ( 演算先レイヤの画像位置における ) ピクセル単位で、仮想座標系において指定します。
vleft 　src 　演算元のレイヤオブジェクトを指定します。
vleft 　sleft 　演算する矩形の左端位置を ( 演算元レイヤの画像位置における ) ピクセル単位で指定します。
vleft 　stop 　演算する矩形の上端位置を ( 演算元レイヤの画像位置における ) ピクセル単位で指定します。
vleft 　swidth 　演算する矩形の横幅を ( 演算元レイヤの画像位置における ) ピクセル単位で指定します。
vleft 　sheight 　演算する矩形の縦幅を ( 演算元レイヤの画像位置における ) ピクセル単位で指定します。
vleft 　mode 　演算のモードを指定します。
vleft 　opa 　演算の強度 ( 0 〜 255 ) を指定します。
戻り値
なし (void)
